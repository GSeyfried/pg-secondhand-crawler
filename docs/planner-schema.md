# Acquisition planner data

The default plan targets September 30, 2026, a USD 3,000 ceiling, and an 82–88 kg estimated all-up weight range. Required components are wing, harness, reserve, and helmet; accessories never block core completion.

Listing history is append-only by stable listing ID. Every observation records its timestamp, asking price, source content hash, location, seller, and documentation signals. Removal is represented as `removed-possibly-sold` with `unknown-final-price`; disappearance is never converted into a confirmed sale.

Market statistics state their sample size and strength. They exclude obvious damaged/parts-only records and IQR outliers when the sample supports it. Prices remain asking prices. Currency conversion is not performed.

Acquisition scenarios report current asking-price subtotal, committed spend, known shipping, known inspection/repack costs, contingency, minimum projected total, unresolved costs, missing components, concerns, and confidence. Unknown values remain unknown. Compatibility and safety claims always require manual confirmation.
